# Frontend Overview

The frontend is a Next.js application using TypeScript and Tailwind CSS. It provides the user interface for interacting with the trading platform.

## Key Components
- **app/**: Main application logic and routing.
- **components/**: Reusable UI components.
- **hooks/**: Custom React hooks for state and effects.
- **lib/**: Utility libraries.
- **store/**: State management (Redux or similar).
- **types/**: TypeScript type definitions.

## Configuration
- **next.config.js**: Next.js configuration.
- **tailwind.config.js**: Tailwind CSS setup.
- **tsconfig.json**: TypeScript configuration.
