# Kimi Docs

This folder documents the TradeCraft project: architecture, backend, frontend, data, infra, and workflows. Use this as an entry point to navigate individual topic docs.

## Files
- architecture.md — high-level system design and data flows
- backend.md — FastAPI backend modules, services, models, schemas, brokers
- frontend.md — Next.js/Tailwind app structure, hooks, components, pages
- data-and-ml.md — feature engineering, models, paper trader logic
- infra-and-ops.md — Docker, env vars, scripts, tasks
- api-surface.md — key REST endpoints (paths, auth, payloads)
- workflows.md — common tasks: run locally, run paper trader, seed/reset
